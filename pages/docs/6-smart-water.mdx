# 6 - SmartWater - Tecnologias IoT para automação de processos
**Autor**: Matheus Eduardo Hoeltgebaum Pereira<br />
**Data**: 19/07/19

Apresentação do trabalho de conclusão de curso desenvolvido para medir, estimar, aferir a quantidade de água passando dentro de um cano.
Foi apresentada uma rede de comunicação IoT que era utilizada para fazer a comunicação com o serviço WEB que armazenava e processava os dados.
